再設計したポジショニング

入口

熱い道具を持たせない

従来ホットカーラーの不安は、カール性能以前に「熱いものを髪に巻く怖さ」。本製品は装着時の心理障壁を下げる。

制御

部位別に温度を変える

前髪・トップ・サイドで必要な温度を変え、毎朝の失敗を減らす。ただしアプリ依存は避け、物理スタートを併用する。

道具

棚に置ける美容家電

ステンレス、白磁、ウォームウッド調を基本に、洗面台やサロン受付に置けるプロダクト感を作る。

最重要論点：髪をどう固定するか

この商品の成否は、アプリでも温度制御でもなく「巻いた髪が自然にほどけない固定機構」で決まる。 クリップを大きく付けると既存ホットカーラーに戻ってしまう。一方で、髪を内側スリットへ通す構造は毛束がよじれる。v2.1では「髪を内側へ入れない」「外側表面で巻き付ける」「風または表面構造でほどけを防ぐ」方針に変更する。
P0 / NG

内側スリット案は保留

毛先を穴や溝に差し込むと、巻き始めで髪がねじれる。髪を通す構造ではなく、外周面に沿って自然に寝かせる構造へ寄せる。

P0 / Air

風で巻き付ける

Dyson Airwrapのバレルが示す通り、髪を風で引き寄せて外周へ巻く技術は存在する。HEJ側の課題は「巻けるか」ではなく「巻いた後、軽いカーラーとして保持できるか」。

P0 / Surface

表面で保持する

外周の浅いスパイラルガイド、低摩擦の山、柔らかい帯で髪を外側に寝かせたまま保持する。絡ませず、ほどく時に引っかけない。

Route A

Air-assist wrap

細い縦スリットを持つバレル外周へ、常温/冷風で髪を引き寄せる。Airwrap型の気流技術は成立しているため、HEJでは「送風源をどこに置くか」と「巻いた後の保持」を分けて設計する。

利点：髪をねじらず外周に誘導できる。クリップレスに近づく。
課題：Dysonとの差別化、知財確認、送風源、騒音、重量、電池消費。
設計分岐：本体内蔵ファン / 別体ハンドル送風 / スタンド送風で巻いてから脱着。
Route B

Surface-hold wrap

風を使わず、外周の形状だけで髪を逃がさない。クリップレスに近いが、髪質差への対応が難しいため、微細構造の試作が必要。

利点：軽い、静か、電池消費が少ない、持ち運びしやすい。
課題：細毛・太毛・レイヤーで保持力が変わる。
検証：浅いスパイラル山、シリコン帯、表面粗さの比較。
Route C

Self-contained curler

1本のカーラー内に小型ファン、風路、電池、制御基板を入れて完結させる案。冷風切替だけなら成立可能性はある。ペルチェ冷却まで入れると排熱と重量が一気に重くなる。

利点：単体で使える。4本それぞれを独立制御できる。
課題：直径、重量、電池持ち、騒音、吸気口の髪巻き込み防止。
現実ライン：冷風=ヒーターOFF常温風。能動冷却=ペルチェは上位/大型モデル候補。

Dyson実物からの示唆

確認

細い縦スリットで巻く

Airwrapのカーラーは、大きなクリップではなく、バレル周方向の気流で髪を外側に巻き付ける。HEJ案も「内側に髪を入れる」のではなく、外側に寝かせて巻くべき。

差分

Dysonはアタッチメント

Airwrapのバレルは送風ハンドルに接続するアタッチメント。HEJの4本カーラーを独立電池式にすると、各本にファンを入れる必要があり、重量と騒音が増える。

設計仮説

巻く機構と保持体を分ける

現実的には「送風で巻く器具」と「髪に残る軽いカーラー」を分ける案が強い。巻いた後に薄い保持スリーブや表面ガイドで形を維持する。

冷却家電リサーチと内部構造案

冷やす技術は既に複数ある。 ただし、カーラーに入れる場合は「冷風」「接触冷却」「送風巻き付け」を混同しない。ドライヤーの冷風は基本的にヒーターOFFの常温風。首冷却家電はペルチェで冷えるが、反対側の熱を逃がすヒートシンクとファンが必要になる。
Cold Air

ドライヤー型の冷風切替

小型ブロワーで風を流し、ヒーターをOFFにして常温風へ切り替える。軽く作れるが、室温以下には冷えない。

構成：BLDCファン、風路、PTCヒーター、温度センサー。
利点：軽い、安い、髪を乱しにくい。
弱点：夏場の洗面所では「冷たい」より「ぬるい風」になりやすい。
Peltier

首冷却家電型の接触冷却

ペルチェ素子で冷却プレートを冷やす。首冷却機のように局所冷却はできるが、熱を反対側へ逃がす設計が重い。

構成：ペルチェ、冷却プレート、ヒートシンク、排熱ファン。
利点：室温以下の冷感を作れる。
弱点：重い、電池消費、結露、髪への熱移動効率。
Hybrid

HEJ向け構造分岐

一体完結カーラーは可能。ただし、軽さを優先するなら送風・冷却源をハンドル/スタンド側に逃がす。単体完結を優先するなら冷風切替までを本体内蔵し、ペルチェは慎重に扱う。

巻く：細い縦スリットから常温/冷風で外周へ誘導。
保持：外周ガイド、薄い保持スリーブ、または低風量キープで固定。
冷やす：本体内蔵冷風、別体ハンドル送風、スタンド送風の3案を比較。
HEJカーラーに入れる場合の内部構造比較
案 内部構造 成立しやすさ 主なリスク
1. カーラー単体ファン内蔵 各カーラーに小型ファン、風路、電池、制御基板、温度センサーを入れる。 可能。 冷風切替までなら一体完結モデルとして成立候補。 重い、太い、騒音、電池持ち、排熱。
2. 別体ハンドル送風 ハンドルにファン/ヒーター/冷風切替を集約。軽量バレルを差して巻き、巻いた後に外して髪に残す。 本命。 Dyson型技術を応用しつつ、保持カーラーで差別化できる。 着脱機構、巻いた後の保持、知財確認。
3. 送風スタンド式 スタンドにファン、必要ならペルチェ/ヒートシンクを置き、カーラーへ冷風を送る。 カーラーを軽くできる。旅行性も作りやすい。 装着しながら巻けるか、使い方が直感的か。
4. ペルチェ内蔵カーラー 各カーラーにペルチェ、冷却面、排熱面、ファン、電池を入れる。 上位モデル候補。室温以下の冷感は作れるが、標準4本セットには重い可能性が高い。 結露、排熱、重量、コスト、髪への冷却効率。
5. 一体完結・冷風モデル 各カーラーに小型ファン、吸気口、縦スリット風路、ヒーターOFF冷風モード、低温ヒーターを入れる。 最初に検討すべき一体型。 ペルチェなしなら重量を抑えやすい。 吸気口の配置、髪巻き込み防止、バッテリー、騒音。

v2仕様仮説

最優先

クリップレス寄り固定

毛先を内側へ通す構造は避ける。風で巻くRoute A、外周表面で保持するRoute B、一体完結カーラーのRoute Cを比較する。

必須

サイズ可変

30 / 32 / 34mmの外径切替は、巻く前にセットする方式を前提にする。髪を巻いた後に径を変える構造は複雑化しすぎる。

要検証

長さ可変

前髪用の短尺、サイド用の標準長を分けるか、伸縮式にするかを検討する。伸縮式は強度・配線・加熱ムラのリスクが高い。

必須

重量と装着感

目標は1本70〜85g台。重いと落下し、軽すぎると電池・発熱部材が成立しない。固定機構と重量は同時に検証する。

必須

持ち運び

4本＋スタンド＋ポーチで旅行に入るサイズにする。ポーチ内でカーラーが動かず、接点や外周リブが傷まない仕切りが必要。

必須

アプリなしでも使える

40代以降の初回離脱を防ぐため、本体またはスタンドに「開始」「温度」「残量」の最低限UIを持たせる。

必須

温度フェイルセーフ

上限温度、過熱停止、肌接触時の熱感、温度ムラをテスト項目化する。美容師監修だけでは安全根拠にならない。

必須

4本同時の成立性

1本ずつ使用なら所要時間が伸びる。訴求は「4本同時使用時」と明記し、1本運用時の目安も併記する。

外部提案前の判断ゲート

固定機構の一次モック

内側スリット案はNG寄り。風で巻くRoute A、表面で保持するRoute B、一体完結Route Cを、3Dプリントまたは簡易モックで比較する。

サイズ・重量・携帯性の実測目標

外径30 / 32 / 34mm、標準長、短尺モデル、1本70〜85g台、4本＋スタンド＋ポーチの収納寸法を仮置きする。

技術成立性の一次確認

ヒーター、電池、充電スタンド、BLE、温度制御、物理操作、発熱安全をOEMに質問できる粒度へ落とす。

原価・MOQ仮説

売価19,800円と24,800円で、製造原価、物流、保証、販促、粗利が成立するか確認する。
